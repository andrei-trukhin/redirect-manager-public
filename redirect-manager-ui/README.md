# Redirect Manager UI

The **Redirect Manager UI** is the administrative web dashboard for the **Redirect Manager** platform. It provides a user-friendly interface to manage URL redirection rules, user accounts, and API tokens.

## Features

- **Dashboard**: Overview of current redirect rules.
- **Redirect Management**: Create, edit, and delete HTTP redirects (301, 302, etc.).
- **User Management**: Manage administrative users who can access the platform.
- **API Token Management**: Generate and manage API tokens for programmatic access to the Redirect Manager API.
- **Responsive Design**: Built with Tailwind CSS for a seamless experience across devices.

## Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
